## Section 1: Login OCI and prepare for deployment

### Step 1.1: Access OCI console

First, let's login OCI console using link below. Username/password details will be provided during lab session. 

[Access OCI  Console](https://cloud.oracle.com)

Use "*fortinetoraclecloud1*" as cloud account name, click *Next*.

![image](https://raw.githubusercontent.com/ozanoguz/fgt-oci-hol/main/images/01.login_oci_1.jpg)

Choose identity domain named "ksa_domain" from the dropdown list of domains.

![image](https://raw.githubusercontent.com/ozanoguz/fgt-oci-hol/main/images/02.login_oci_2a.JPG)

Enter username and password, then click "Sign In". 

![image](https://raw.githubusercontent.com/ozanoguz/fgt-oci-hol/main/images/03.login_oci_3a.JPG)
